Reflections

Going into this class I was slightly skeptical on what I would get out of it, since I had already completed a few different lessons in HTML/CSS from Khan Academy a couple years back. However, I found that there is a great deal more to web development than just typing some words and marking it up so it formats well on a page. I learned about the overall development lifecycle of websites. From the introductory site structure plan to the scripting languages behind the scenes in the servers. Everything really started to click for more when we went over the client-server interactions with the different scripts, protocols, and gateways that make the internet possible. I discovered that I knew so little prior to taking this course, but now, I feel that I can successfully design and create a website. Which will be much more dynamic when I learn more about Javascript and other scripting languages.
